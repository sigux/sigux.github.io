Github Pages + HEXO + Node.js搭建个人博客

系统准备

1.安装Git

Installing Hexo is quite easy and only requires the following beforehand:

  • Node.js(Should be at least Node.js 10.13, recommends 12.0 or higher)
  • Git
  1. 安装Node.js

    If you use the official installer, make sure Add to PATH is checked (it’s checked by default).

  1. 安装HEXO
    在桌面右键，点击Git Bash Here。HEXO会显示Git安装语句 $ npm install -g hexo-cli。
    屏幕显示
    + hexo-cli@0.0.0格式，说明安装成功。
    退出Git。

  2. 建立个人博客文件夹
    新建个人博客文件夹如blog，确保文件夹为空。首先，进行blog文件夹初始化，打开文件夹，点击右键，选择Git Bash Here，输入
    hexo init
    初始化完成后，blog文件夹中会有相应的文件目录，不要变动，继续输入
    hexo install
    安装成功后，继续输入（安装插件），
    npm install hexo-deployer-git --save
    显示+ hexo-deployer-git@3.0.0表示安装成功。

  3. 打开Github，创建github.io仓库
    创建username.github.io仓库，点击Clone菜单，选择SSH，复制内容。回到初始化并安装好hexo的博客文件夹，使用Notepad打开config.yml。
    滚动至文件末尾deployment，填写如下内容：

    type: git
    repo: git@github.com:username/username.github.io.git
    branch : master

  1. 部署hexo博客至github
    在git中输入
    hexo g -d
